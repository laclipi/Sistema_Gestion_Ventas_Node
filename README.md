# Gestión_Ventas_Node
# Sistema de Gestión de Ventas

Aplicación web fullstack para la gestión de productos, clientes y facturación.

## Tecnologías
- Node.js
- Express
- MySQL
- HTML
- CSS
- JavaScript

## Funcionalidades
- CRUD de productos
- Registro de clientes
- Gestión de ventas y facturación
- Arquitectura separada frontend / backend

## Estado
Proyecto funcional, preparado para ampliaciones.

## Aprendizajes
- Desarrollo de API REST con Node.js y Express
- Modelado de datos y conexión con MySQL
- Separación de responsabilidades entre frontend y backend
